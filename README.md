# tmng-pptx-linter

WIP :sunglasses:

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tmng-pptx-linter'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tmng-pptx-linter

## Usage

```bash
$ tmng-pptx-linter check /path/to/tmng.pptx # checking pptx file.
```
